# Follow the progress - FTP

### Configurações Iniciais

- Crie um arquivo .env na raiz do projeto.
- Verifique as variáveis no arquivo .env.example
- Instale as dependências
- Inicie o projeto

<span style="color:#C5A66E">Terminal</span>

```console
~$: yarn install
~$: yarn dev
```
